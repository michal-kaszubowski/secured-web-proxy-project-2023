Welcome to my API!

Here is a small list of what endpoints are currently aviable for this API:

  • / [GET, POST] shows info about API; enables to test HTTP requests.

  • /check

    • /user [GET] checks if supported access token enables access to user API scope.
    • /admin [GET] checks if access token grants permission to access admin API scope.

  • /user

    • /public/data [GET] returns data accessible for all loged in users.
    • /private/:userID [GET] returns data for the specific user.

  • /admin

    • /public/data [GET] returns data visible for all admins.
