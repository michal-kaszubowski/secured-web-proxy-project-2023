Welcome to my Confidential Proxy Node.js Server for my OAuth2.0 secured Single Page Application!

Here is a small list of what endpoints are currently aviable for this API:

  • / [GET, POST] shows info about API; allows to test HTTP requests.

  • /oauth

    • / [GET] validates connection with the Authorization Server.
    • /init?code={authorization_code_here} [GET] proxy redemes token and stores it safly.
    • /userinfo [GET] gets data about currently logged in user.
    • /end [GET] logs out currently logged in user.
  • /forward?request={userpublic || userprivate || adminpublic} forwards request to the API automatically refreshing token if needed.
