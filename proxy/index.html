Welcome to my Confidential Proxy Node.js Server for my OAuth2.0 secured Single Page Application!

Here is a small list of what endpoints are currently aviable for this API:

  • / [GET, POST] shows info about API; enables to test HTTP requests.
  • /oauth/flow/check [GET] validates connection with the Authorization Server
  • /oauth/flow/init?code={authorization_code_here} [GET] starts oauth token redeeming process.
